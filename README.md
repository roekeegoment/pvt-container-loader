# PVT Container Loader

**Interactive 3D Container Load Calculator** — PVT Foods Industries Co., Ltd.

🔗 **Live App:** https://roekeegoment.github.io/pvt-container-loader/

## Modes
- 📦 **Single Load** — 6 loading patterns (3+3+2, 6+2, Uniform A/B, Upright A/B)
- 🏗 **Pallet Load** — EUR/EPAL, US/GMA, Thai Standard pallets
- 🎨 **Mix Load** — up to 4 SKUs in one container with color-coded 3D view
- 🏭 **Warehouse** — pallet design with dynamic load safety limits

## Features
- 29 SKUs preloaded with box dimensions and GW
- 3D visualization — drag to rotate, scroll to zoom
- PDF report with Front View + Side View grids
- Supports 20ft, 40ft, 40ft High Cube containers
- Gross weight vs payload check
